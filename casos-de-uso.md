# Casos de Uso – Fase 1

## 1. Autenticar-se
1. Usuário informa login e senha.
2. Sistema valida credenciais.
3. Acesso concedido.

## 2. Manter Alunos
1. Seleciona “Manter Alunos”.
2. Sistema solicita dados pessoais e acadêmicos.
3. Extende: Pessoa Física.
4. Inclui: Validar CPF, Endereço, Contatos, Documentos.
5. Sistema salva.

## 3. Manter Professores
1. Seleciona “Manter Professores”.
2. Sistema solicita dados pessoais, formação e documentos.
3. Extende: Pessoa Física.
4. Inclui: Validar CPF, Endereço, Contatos, Documentos.
5. Sistema salva.

## 4. Manter Fornecedores
1. Seleciona “Manter Fornecedores”.
2. Sistema solicita dados da empresa.
3. Extende: Pessoa Jurídica.
4. Inclui: Validar CNPJ, Endereço, Contatos, Documentos.
5. Sistema salva.

## 5. Manter Pessoa Física
1. Informa/edita dados pessoais.
2. Valida CPF, endereço, contatos, documentos.
3. Sistema salva.

## 6. Manter Pessoa Jurídica
1. Informa dados da empresa.
2. Valida CNPJ, endereço, contatos, documentos.
3. Sistema salva.

## 7. Manter Endereço e Contatos
1. Usuário informa endereço e contatos.
2. Sistema valida.
3. Sistema salva.

## 8. Anexar/Validar Documentos
1. Usuário envia documentos.
2. Sistema valida formato e integridade.
3. Sistema salva.

## 9. Validar CPF
1. Sistema recebe CPF.
2. Valida dígitos.
3. Retorna resultado.

## 10. Validar CNPJ
1. Sistema recebe CNPJ.
2. Valida dígitos verificadores.
3. Retorna resultado.
